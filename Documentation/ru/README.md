## 🛠 Что такое s&box?

**s&box** — это игровой движок нового поколения от Facepunch (создателей Garry's Mod). Он использует:

- **C#** — основной язык программирования
- **Компонентную систему** — как Unity, но проще
- **Razor** — для создания UI (интерфейса)
- **SCSS** — для стилизации UI
- **Встроенный мультиплеер** — сетевая игра из коробки

## 📂 Структура типичного проекта s&box

```
МойПроект/
├── code/                    # Весь код проекта
│   ├── Components/          # Компоненты (игровая логика)
│   │   ├── Player/          # Компоненты игрока
│   │   ├── Weapons/         # Компоненты оружия
│   │   └── World/           # Компоненты мира
│   ├── UI/                  # Пользовательский интерфейс
│   │   ├── Hud.razor        # HUD - Razor шаблон
│   │   ├── Hud.razor.scss   # HUD - стили (ВАЖНО: .razor.scss!)
│   │   ├── Menu.razor       # Меню
│   │   └── Menu.razor.scss  # Стили меню
│   └── Assembly.cs          # Глобальные настройки
├── Assets/                  # Ассеты (модели, текстуры, звуки)
│   ├── Models/
│   ├── Materials/
│   └── Sounds/
├── ProjectSettings/         # Настройки проекта
└── .addon                   # Файл конфигурации аддона
```

> ⚠️ **ВАЖНО о файлах стилей:** Движок s&box **НЕ видит** обычные файлы `.scss`. Стили для Razor-компонентов должны иметь расширение **`.razor.scss`**! Например: `MyPanel.razor` + `MyPanel.razor.scss`.
